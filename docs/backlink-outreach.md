## Outreach Workflow

### Per-Site Checklist
1. Find the best contact page or editor email
2. Read 3+ articles to tailor the pitch
3. Send from a personal domain (not generic Gmail)
4. Follow up once after 10 days if no response
5. If accepted, prepare the content/link within one week

### Link Placement Priority
| Type | Value | Effort | Priority |
|---|---|---|---|
| .edu resource page | ★★★★★ | Medium | High |
| Industry blog guest post | ★★★★ | High | High |
| Tool directory listing | ★★★ | Low | Medium |
| GitHub awesome list | ★★★ | Low | Medium |
| Forum signature/profile | ★ | Low | Low |

### Tracking Spreadsheet Columns
```
Site URL | Contact | Email Sent | Follow-up | Status | Link URL | Notes
```
